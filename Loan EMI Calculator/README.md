# 💰 Loan EMI Calculator

A simple and professional **Loan EMI Calculator** built with **Python**. This application calculates the **Monthly EMI (Equated Monthly Installment)**, **Total Interest Payable**, **Total Payment**, and displays an optional **Amortization Schedule**.

---

## 🗒️ Description

The Loan EMI Calculator helps users estimate their monthly loan repayments based on the loan amount, annual interest rate, and loan tenure. It also provides a detailed repayment schedule showing the principal, interest, and remaining balance for each month.

---

## ✨ Features

- 💰 Calculate Monthly EMI
- 📈 Calculate Total Interest Payable
- 💵 Calculate Total Amount Payable
- 📅 Display Amortization Schedule
- ✅ Input Validation
- ⚡ Fast and Lightweight
- 💻 Command-Line Interface
- 📚 Beginner-Friendly Code
