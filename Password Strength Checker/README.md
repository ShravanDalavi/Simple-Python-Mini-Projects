# 🔐 Password Strength Checker

This repository contains a simple **Password Strength Checker** application built using **Python**. The application analyzes the strength of a password based on common security rules and provides suggestions to improve it.

---

## 🗒️ Description

This project checks whether a password is secure by verifying:

- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character

The program then rates the password as:

- 🔴 Very Weak
- 🟠 Weak
- 🟡 Medium
- 🟢 Strong
- 🟢 Very Strong

---

## 📋 Required Modules

This project uses only Python's built-in library:

- `re`

No external libraries are required.

